# Tank-Level-SCADA
Benchtop IIoT level monitoring system. An ESP32 reads an HC-SR04 ultrasonic sensor, publishes the tank level over Modbus TCP, and streams it to a live Ignition (Perspective) SCADA dashboard with ISA-101 high-performance HMI design and historian trending. Phase 1: Monitoring. Phase 2 (control) in development.
